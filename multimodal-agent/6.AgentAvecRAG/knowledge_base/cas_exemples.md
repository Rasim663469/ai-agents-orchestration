# Cas Exemples Pour L Agent

## Cas 1: Facade Avec Fissure Fine
Observation:
- Une petite fissure verticale est visible sur une facade.

Reponse attendue:
- Signaler la fissure comme anomalie visible.
- Preciser qu'elle semble localisee.
- Eviter d'affirmer un danger structurel si l'image ne le prouve pas.

## Cas 2: Toiture Degradee
Observation:
- Plusieurs tuiles paraissent absentes ou deplacees.

Reponse attendue:
- Mentionner la degradation visible.
- Signaler un risque potentiel d'infiltration.
- Indiquer qu'une verification sur place serait utile.

## Cas 3: Traces D Humidite
Observation:
- Presence de taches sombres sur une paroi.

Reponse attendue:
- Decrire les traces.
- Evoquer une humidite possible sans la presenter comme certaine.
- Recommander une verification complementaire si necessaire.

## Cas 4: Image Incomplete
Observation:
- Une partie importante de la scene n'est pas visible.

Reponse attendue:
- Decrire ce qui est visible.
- Signaler les limites de l'image.
- Refuser toute conclusion trop forte sur les zones absentes.
